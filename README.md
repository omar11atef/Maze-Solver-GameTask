# Maze Solver Game using Intelligent Agents

An AI-based Maze Solver Game that compares different Intelligent Agents inside a maze environment.

## Features

* Learning Agent using Q-Learning
* Goal-Based Agent using A* Search
* Interactive GUI using HTML Canvas
* Maze generation
* Agent movement animation
* Difficulty levels
* Performance comparison
* Statistics panel
* Replay and reset system

---

# Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Canvas API

Algorithms:

* Q-Learning
* A* Search Algorithm

---

# Intelligent Agents

## Learning Agent

Uses Q-Learning to learn the best path through trial and error.

Features:

* Q-table
* Reward system
* Exploration vs exploitation
* Training mode
* Inference mode

---

## Goal-Based Agent

Uses A* Search Algorithm to find the shortest deterministic path.

Features:

* Heuristic function
* Path reconstruction
* Node exploration
* Shortest path search

---

# How to Run

## Option 1: Open Directly

Simply open:

```bash
index.html
```

in your browser.

---

## Option 2: Run Local Server

Using Python:

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```
